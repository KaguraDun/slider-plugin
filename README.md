# JQuery Slider plugin

[Example](https://kd-slider-plugin.netlify.app/example)
Slider with MVP architecture. Data transfer by Observer pattern.
### Main features
* Change step
* Vertical / horizontal view
* Single value / range
* Interface for change slider values
* Interactive scale of values
* Progress bar
* Value tip
* Toggle elements: tip, bar, scale 
* Work with touch devices
* Responsive
## API
### Usage Example
```js
// import jQuery and slider from dist
   import $ from  'jquery';
   import './dist/slider';
   
   const sliderSettings  = {
		from: -5000,
		to: 2500,
		min: -5000,
		max: 5000,
		step: 5,
		showBar: true,
		showScale: true,
		showTip: true,
		isRange: true,
		isVertical: true,
	};

	const $slider = $(sliderContainer).createSlider(sliderSettings);

	// chaining
	$slider.setMin(-100).setMax(100);
```
### Settings
|name|type|info|
|--|--|--|
|min|number|required|
|max|number|required|
|from|number|required|
|to|number|--|
|step|number|above 0|
|showBar|boolean|--|
|showScale|boolean|--|
|showTip|boolean|--|
|isRange|boolean|--|
|isVertical|boolean|--|
